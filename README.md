# 🛒 Ecommerce Website

A simple frontend e-commerce website built using **HTML, CSS, and JavaScript**. The project fetches product data from the DummyJSON API, allows users to view product details, and add products to a shopping cart using the browser's local storage.

## 🚀 Features

- 🛍️ Display products from DummyJSON API
- 🔍 View detailed information about each product
- 🛒 Add products to cart
- 💾 Cart data stored using Local Storage
- 📱 Responsive layout
- 🎨 Clean and simple user interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- DummyJSON API

## ⚙️ How to Run

1. Clone the repository

```bash
git clone https://github.com/amanmani893/Ecommerce-website.git
```

2. Open the project folder.

3. Open `index.html` in your browser.

Or visit the GitHub Pages deployment (if enabled).
